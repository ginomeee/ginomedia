# ginomedia

Personal website and CV landing page for [gino.media](https://gino.media).

A hobby frontend project — the goal was to design something that makes a CV easy to understand at a glance, rather than handing someone a PDF. Built with a dark, tech-centered aesthetic using card-based layouts, hover interactions, and a grid background.

## Stack

- **Jekyll** — static site generator
- **GitHub Pages** — hosting
- **HTML / CSS / JavaScript** — no frameworks, no build tools

## Running locally

```bash
gem install bundler jekyll
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000`.

## Notes

This is a personal hobby project and not intended as a template or starter kit. Design and content are specific to my own CV and background.
